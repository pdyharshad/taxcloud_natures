TaxCloud Exemption (Sale)

Description

This module automatically applies a valid exemption certificate on Sales Orders, Website Orders, and Subscription Orders based on the Delivery Address selected. It is compatible with multi-company environments.

Configuration

The main module, TaxCloud Exemption, needs to be installed.

The configurations and Access Rights are detailed in the documentation of that module, please refer to it for clarification.

Functionality

TaxCloud rates are used on a sales order/subscription order if the proper fiscal position is selected, by default it is named "Automatic Tax Mapping (TaxCloud)”.  The TaxCloud tax calculation is triggered when the Send By Email, Update Taxes, or Confirm button is clicked.

A valid exemption associated with the Delivery Address or its parent Company will be automatically applied to the sales order/subscription order at the time of the Tax calculation. The validity of the exemption will be checked based on the sales order Date. A  message “Exemption is applied” appears in the header.

The Exemption applied is shown under the Other Info tab. The Don't Apply Exemption Boolean can be used if the exemption does not need to be applied.

A valid Exemption will also be applied automatically to Website Orders after the Shipping/Delivery address is selected.

When an Invoice is created from the Sales Order, the Exemption and the Don't Apply Exemption Boolean will be propagated to the Invoice.

On the Exemptions, a smart button for Sale Orders has been added that will list all the Sales Orders associated with the corresponding Exemption.
